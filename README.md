# fauxdelio_testing

Perl Docker image with POE and other modules

### Build it
```bash
host > docker build -t perls:poe .
```

### Run it
```bash
host > bash runner.sh
```

Presently deployed to mtn-corpinteg.mtnsat.io port 2019
